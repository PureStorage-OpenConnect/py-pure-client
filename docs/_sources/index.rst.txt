Overview
--------

This Python package provides clients that use the Pure1, FlashArray, and FlashBlade REST APIs.

:doc:`installation`
  Steps for installing the Python package.

:doc:`quick_start`
  A quick start guide for the Python clients.

:doc:`fa_reference`
  A glossary of all interfaces and models used by the FlashArray client.

:doc:`fb_reference`
  A glossary of all interfaces and models used by the FlashBlade client.

:doc:`pure1_reference`
  A glossary of all interfaces and models used by the Pure1 client.


Requirements
------------

The library requires Python 2.7 and higher or Python 3.3 and higher. Third-party libraries are also required.

Community
---------

To learn what other Pure Storage customers are doing with the client, to share
an implementation or contribute code, or to interact with Pure Storage,
visit the `Pure Storage Community <http://community.purestorage.com/>`_.
